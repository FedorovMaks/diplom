# Система опросов и анкетирования АНО ПО «ПГТК»

Веб-приложение для проведения опросов студентов и преподавателей колледжа.

## Требования

- Node.js версии 18 или выше

## Установка и запуск

```bash
# 1. Установить зависимости
npm install

# 2. Настроить переменные окружения
# Скопировать .env.example в .env и заполнить свои данные
cp .env.example .env

# 3. Создать таблицы в базе данных
npm run init-db


# 4. Запустить сервер
npm run dev
если node js версии 17 и младше то
node server.js
# 5. Открыть в браузере
# http://localhost:3000
```

## Переменные окружения (.env)

| Переменная       | Описание                                                                                               |
| ---------------- | ------------------------------------------------------------------------------------------------------ |
| `DATABASE_URL`   | Строка подключения к PostgreSQL (Supabase → Settings → Database → Connection string, Transaction mode) |
| `ADMIN_LOGIN`    | Логин администратора                                                                                   |
| `ADMIN_PASSWORD` | Пароль администратора                                                                                  |
| `SESSION_SECRET` | Секретный ключ для подписи cookie (минимум 32 символа)                                                 |
| `PORT`           | Порт HTTP-сервера (по умолчанию 3000)                                                                  |

## Структура проекта

```
survey-app/
  server.js              — точка входа Express-сервера
  package.json           — зависимости и скрипты
  .env                   — переменные окружения
  routes/
    surveys.js           — API опросов (POST)
    teachers.js          — API преподавателей (GET)
    admin.js             — авторизация и экспорт Excel
  db/
    pool.js              — подключение к PostgreSQL
    schema.sql           — SQL-схема базы данных
  public/
    css/style.css        — стили интерфейса
    js/groups.js         — список групп колледжа
    js/survey-general.js — логика общего опроса
    js/survey-pps.js     — логика оценки ППС
    js/survey-teacher.js — логика опроса преподавателей
    index.html           — главная страница
    student.html         — раздел студентов
    survey-general.html  — общий опрос
    survey-pps.html      — оценка ППС
    survey-teacher.html  — опрос преподавателей
    admin-login.html     — вход администратора
    admin.html           — выгрузка результатов
    thanks.html          — подтверждение отправки
  data/
    groups.js            — конфигурация групп
    teachers.json        — список преподавателей
  scripts/
    init-db.js           — инициализация БД
```

## Виды опросов

- **Общий опрос студентов** — 10 вопросов об удовлетворённости обучением
- **Оценка ППС** — 12 характеристик для каждого преподавателя по 5-балльной шкале
- **Опрос преподавателей** — 6 вопросов об условиях работы (анонимный)

## Администрирование

1. Перейти на http://localhost:3000/admin-login.html
2. Ввести логин и пароль из .env
3. Скачать результаты опросов в формате Excel (.xlsx)
